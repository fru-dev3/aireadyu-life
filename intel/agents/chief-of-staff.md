# Life Operations Director — Setup

## Connect External Instructions

1. **Download the AI Ready Life: Intel pack** from [Gumroad](https://frudev.gumroad.com/l/aireadylife-intel)
2. **Extract to** `~/Documents/AIReadyLife-Intel/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced → Switch from Managed to External**
5. **Paste path:**

```
~/Documents/AIReadyLife-Intel/agents/chief-of-staff
```

The agent will load full instructions from `~/Documents/AIReadyLife-Intel/agents/chief-of-staff/AGENTS.md`
