# Life Operations Director — Setup

## Connect External Instructions

1. **Download the AI Ready Life: Real Estate pack** from [Gumroad](https://frudev.gumroad.com/l/aireadylife-realestate)
2. **Extract to** `~/Documents/AIReadyLife-RealEstate/`
3. **In the Paperclip dashboard**, click on this agent
4. **Click Advanced → Switch from Managed to External**
5. **Paste path:**

```
~/Documents/AIReadyLife-RealEstate/agents/chief-of-staff
```

The agent will load full instructions from `~/Documents/AIReadyLife-RealEstate/agents/chief-of-staff/AGENTS.md`
